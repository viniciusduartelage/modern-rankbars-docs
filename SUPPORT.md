# Support — Modern RankBars

Need help with **Modern RankBars** (Power BI custom visual)?

## How to get support

- **Report a bug or request a feature:** open an issue at
  <https://github.com/viniciusduartelage/modern-rankbars/issues>
- **Email:** `nic@orthocrin.com.br`
- **Documentation:** see the [README](README.md) (fields, formatting options, how to import).

Support is provided on a best‑effort basis. When reporting a problem, please include your
Power BI version, what you expected, what happened, and a screenshot if possible.

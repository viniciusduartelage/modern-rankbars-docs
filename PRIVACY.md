# Privacy Policy (Modern RankBars)

Last updated: 2026-07-01

Modern RankBars is a custom visual for Microsoft Power BI.

## Data collection

Modern RankBars does not collect, store, or transmit any data.

- The visual makes no external network requests of any kind. There is no HTTP or HTTPS traffic,
  no WebSocket, no third party service, and no CDN.
- The data you place in the visual's fields (category, value, details) is used only to render the
  ranking inside your own Power BI report, on your machine or tenant, and it never leaves Power BI.
- There are no cookies, no tracking, no analytics, and no telemetry.
- The author has no access to any data shown in the visual.

All processing happens locally within the Power BI rendering sandbox.

# Política de Privacidade (Modern RankBars, pt-BR)

O Modern RankBars é um visual personalizado para o Microsoft Power BI.

Ele não coleta, não armazena e não transmite nenhum dado. O visual não faz nenhuma requisição
externa; os dados colocados nos campos (categoria, valor, detalhes) são usados apenas para
desenhar o ranking dentro do seu próprio relatório do Power BI, e nunca saem do Power BI. Não há
cookies, rastreamento, análise ou telemetria. O autor não tem acesso a nenhum dado exibido no
visual.

Para dúvidas, abra uma issue em
https://github.com/viniciusduartelage/modern-rankbars-docs/issues ou escreva para
vinicius@dulage.onmicrosoft.com.
